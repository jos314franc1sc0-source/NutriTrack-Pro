# NutriTrack Pro 🥗📊

Aplicación web para la gestión de pacientes y seguimiento nutricional mediante IMC.

## 🚀 Funcionalidades
- Registro de pacientes
- Cálculo automático de IMC
- Clasificación: Bajo peso, Normal, Sobrepeso, Obesidad
- Registro de consultas
- Evolución del IMC en gráfica
- Creación de planes nutricionales
- Login con roles:
  - Nutricionista
  - Paciente
